### BOS_s2sh

- s2sh框架简单实现了BOS系统的部分功能(物流系统)

- 测试数据文件为bos_ssm.sql , 数据库为mysql  编码 utf-8

- 管理员账号密码 u : admin   p : admin

#### 项目构建

	- 使用maven构建项目
	
	- 后端框架使用的S2SH
	
	- 前段框架 EasyUI , zTree , highchars 
	
	- javaemail(java发送邮件)  quarzt(定时任务)
	
	- 安全验证使用的shiro (只设置了登陆验证以及登录以后将权限放到对象中,并没有在方法上进行控制)
	
	- apache POI

	- 对ajax提交表单的数据更熟练

	- js前段图表插件

#### 经验

- 对ajax更加的熟练

- 框架的代码程度上升

- 学习到了shiro

- maven的使用方法
